materialdesignicons F02AF "Coast Guard Client" as client

materialdesignicons F1288 "MLN server" as mln

user->client: Opens game
client->server: get_token()
server->server: No username found\nfor cookie provided
server->client: token
client->user: Prompts login

====

user->mln: Sign in with cookie
mln->server: cookie + username
server->client: Save cookie\nRedirect to game

client->server: get_token()
server->server: Found username\nfor given cookie
server->client: token + username\nwith in-game progress
client->user: Ready to play

====

user->client: Beats a level
client->server: save_score()
server->mln: POST /rank
mln->user: Sends rewards via MLN message
